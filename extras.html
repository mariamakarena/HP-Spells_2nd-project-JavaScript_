Info about the project and the author

This project is a simple web application that displays a list of Harry Potter spells. It uses HTML, CSS, and JavaScript to create an interactive experience for users.

The spells data is live fetched from a public API, and the application dynamically generates random spells, spells cards filtered for the user to search for their favourite spell and a table to display the spell names and their descriptions.

The project was created by mariamakarena, as part of the Web Development course at Code Academy Berlin. May 2025.
